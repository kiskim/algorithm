+ 스터디 방법
    + 삼각형의 모서리에 대한 규칙 탐색
    + 삼각형의 갯수가 n / 3 + (n % 3 > 0 : + 1) 이되는 규칙 화인
    + 코드 구현을 통해 첫 삼각형의 외각을 채우는 프로그램 구현 성공
    + 구현 한 프로그램의 반복을 구현해 문제를 해결을 시도했지만 코드가 복잡해 해당 방법 보류
    + 배열을 통해 해결하는 방식이 있어 해당 방식을 통해 다음 스터디까지 숙제로 남김
